The [Bootstrap Card](https://getbootstrap.com/docs/5.1/components/card/) footer container

#### Usage

```jsx
<Card>
	<CardHeader>
		{name}
	</CardHeader>

	<CardBody>
		{details}
	</CardBody>
    
	<CardFooter>
		<Button onClick={() => save()}>Save</Button>
	</CardFooter>
</Card>
```
