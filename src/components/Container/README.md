The [Bootstrap Container](https://getbootstrap.com/docs/5.1/layout/containers/)

#### Usage

```jsx
<Container>
    ...
</Container>
```

```jsx
<Container size={'sm'}>
    ...
</Container>
```

Use the flex attribute to make the container fit to the full-width

```jsx
<Container flex>
    ...
</Container>
```
