The [Bootstrap Card](https://getbootstrap.com/docs/5.1/components/card/) image with a built-in font-awesome icon

#### Usage

```jsx
<Card>
	<CardIcon icon={'users'} background={'tomato'}/>

	<CardBody>
		{details}
	</CardBody>
    
	<CardFooter>
		<Button onClick={() => save()}>Save</Button>
	</CardFooter>
</Card>
```
